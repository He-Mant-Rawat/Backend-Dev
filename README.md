📖 Journey So Far 🖥️ 5th January 2026 — Unix Commands

I began my backend development journey by learning the fundamentals of Unix commands, a core skill for every backend developer. This phase helped me understand how systems work behind the scenes and how to interact efficiently with the operating system.

🔹 What I learned:

File System Navigation: Using commands like cd, ls, and pwd to move through directories with confidence.

File Management: Creating, deleting, copying, and modifying files using touch, rm, cp, and mv.

Permissions & Security: Understanding file permissions and access control to ensure system safety and proper authorization.

These skills laid a strong foundation for managing servers and backend environments.
